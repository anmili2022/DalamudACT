## 主要更新

### 极简悬浮样式

- 新增 `Minimal / 极简样式`，悬浮窗现在支持 `Classic / Ikegami / Minimal` 三种展示模式。
- 极简样式支持单独配置表头、汇总条、占比条文本、职业/伤害/死亡列，以及行高、字号和列宽。
- 设置页已补齐极简样式对应的独立配置区，并支持按样式恢复默认设置。

### DoT / 特殊伤害统计修正

- 所有 `LB / 极限技` 不再并入玩家常规伤害统计。
- `白魔 / 学者 / 占星术士 / 贤者` 的已知 DoT 已统一切到更保守的估算链路，优先避免虚高。
- 机工 `野火` 现在走独立状态追踪与到期引爆模拟，不再整段漏算。
- 补强带 `OwnerId` 的玩家额外来源归属链，像 `英雄的掠影 / 礼仪之铃 / 后式自走人偶` 这类对象更容易稳定归回主人。

### 离线对账工具

- 新增 `tools/DotReconcile`，可直接读取 `history-records.json` 并扫描 ACT `Network_*.log` 做 DoT 快速对账。
- 默认采用 `hostile-only 24|DoT` 口径，并额外区分 `ACT 已归属` 与 `未归属 hostile DoT`，避免把 ACT 源丢失误判成插件虚高。
- `README.md` 已同步补上工具使用说明、常用命令和排查建议。
