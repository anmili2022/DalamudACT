# DPS统计 发布说明

- 当前发布版本：`0.15.2.2`
- 这份文档会作为 GitHub Release 正文使用。

## 本次更新

- 悬浮窗默认展开尺寸调整为 `300x300`，避免首次打开时过大。
- 插件加载后，悬浮窗默认进入折叠态，不再先显示页签栏。
- 折叠态会显示 `等待战斗数据...`，并与展开态使用同一位置的提示文本。
- 点击 `等待战斗数据...` 可以在展开态与无页签折叠态之间切换。
- 右键折叠态提示文本仍可直接打开设置窗口。

## 发布链修正

- 正式发布 workflow 现在会显式读取 UTF-8 的发布说明文件。
- GitHub Release 会自动带上本地发布说明正文，避免再次出现发布页正文乱码或缺失。

## 重点说明

- 远端已经存在 `0.15.2.1` tag，因此本次版本号提升为 `0.15.2.2`。
- `repo.json`、插件 manifest 和程序集版本号已同步到 `0.15.2.2`。

## 发布后建议检查

1. 确认 GitHub 上生成 `DalamudACT 0.15.2.2` Release。
2. 确认附件中存在 `DalamudACT.zip`。
