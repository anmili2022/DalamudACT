## 未发布

### 2026-05-06

#### 新增

- `DPS` 页面新增 `伤害量` 列
- `伤害量` 列支持和其他列一样单独显示 / 隐藏
- 悬浮面板设置中新增：
  - `玩家列最小宽度`
  - `固定列宽`
  - `表格行高`
- 历史记录现在保存完整战斗快照
- 点击历史记录条目后，可直接回看那场战斗的数据
- 插件补上主窗口入口回调，修复“未注册主 UI 回调”的校验警告

#### 优化

- `DPS` 统计范围收口为：
  - 自己
  - 当前队伍里的队友
- 队伍里的 NPC 队友也会进入统计
- 队伍外玩家不会进入统计
- 伤害相关文本改为中文单位显示：
  - `万`
  - `亿`
  - `兆`
- `DPS` 页签支持：
  - 左键折叠 / 恢复
  - 右键打开 / 关闭设置窗口
- 折叠态窗口尺寸调整为：
  - 宽度 `270`
  - 高度 `42`

#### 历史记录与测试数据

- 历史记录支持高亮当前选中项
- 查看历史记录时，`DPS / HPS / 承伤 / 概览` 会一起切换到那场战斗
- 新战斗开始后，会自动切回实时数据
- `导入测试数据` 不再清空已有历史记录
- 导入测试数据时，相同测试记录改为覆盖更新，不再无限重复追加
- 3 条测试数据已经拆成不同内容的场景快照，不再只是名称不同

## 0.15.0.0 - 2026-05-05

### 核心变化

- 插件定位调整为独立运行的 Dalamud DPS 统计插件
- 旧版依赖外部 `MiniParse` 的统计链路下线
- 改为在游戏内直接采集战斗事件并生成本地统计结果
- 界面与展示模型整体转向 `ACTX / NotACT` 风格

### 兼容与稳定性

- 处理 `ISigScanner / SigScanner` 注入兼容问题
- 处理 `IClientState.TerritoryType` 运行时缺失问题
- `Framework.Update` 增加异常隔离
- 当前仅保留最稳的 `ActionEffect` 统计主链路
- 暂时禁用：
  - `ActorControlSelf`
  - `Cast`

### 界面与交互

- 新增主窗口
- 新增设置窗口
- 悬浮统计面板独立化
- UI 中文化
- 插件名称统一为 `DPS统计`

### 面板能力

- 悬浮面板支持：
  - `DPS`
  - `HPS`
  - `承伤`
  - `概览`
  - `历史记录`
- 支持透明度调整
- 支持按页签显示 / 隐藏
- 支持职业主题色与单色两种占比条模式
- 支持 `DPS` 页面显示人数设置

### 统计细节

- `DPS` 数值口径改为秒伤
- 支持统计队伍中的 NPC 队友
- 职业名改为中文职业名
- `DPS` 页支持：
  - `职业`
  - `秒伤`
  - `死亡`
 这些列的显示控制
- 新增 `总DPS / 全队` 汇总行
- 死亡统计改为通过队伍成员血量变化进行记录
