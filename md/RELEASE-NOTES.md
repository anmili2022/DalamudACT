# DPS统计 发布说明

说明：

- 当前发布版本为 `0.15.2.1`
- 这份文档可以直接作为 GitHub Release 文案基础

## 一句话介绍

`DPS统计` 这次更新重点修复了和 NPC 队友、信赖、小队成员一起战斗时不出统计的问题，同时补齐了历史记录与实时视图切换的几个体验问题。

## 短版更新公告

可直接使用下面这段：

> `DPS统计` 现已修复与 NPC 队友、信赖、小队成员战斗时不出统计的问题，队伍解析改为按实际队伍槽位识别成员；同时优化了历史记录体验，战斗结束后不再停留在历史页，导入历史后会自动打开最新记录，历史记录页也补上了清空历史按钮。主窗口现在会显示插件版本号，方便确认当前加载版本。

## 标准版更新说明

### 1. NPC 队友统计修复

- 队伍识别新增按游戏队伍槽位 `<1> .. <8>` 解析成员
- 修复信赖 / 小队 / NPC 队友战斗时仍然不出统计的问题
- NPC 队友现在会参与：
  - 伤害 / 治疗统计
  - 死亡检测
  - 脱战结束判定

### 2. 历史记录与实时视图体验修复

- 战斗结束后不再自动停留在 `历史记录`
- 导入历史记录后自动打开最新一条
- 历史记录页面新增 `清空历史`
- 当前没有实时战斗但已有历史记录时，会提示可以查看历史

### 3. 其他修正

- 主窗口显示插件版本号
- 时长小于 `30` 秒的战斗不写入历史记录

## 面向玩家的说明

如果你平时主要是和信赖、剧情 NPC 或小队成员一起打本，这一版的意义最大，因为之前最明显的问题就是这些战斗经常没有统计。现在这条链路已经补上，日常查看当前队伍输出、承伤和历史记录会稳定很多。

## 更新后建议

1. 进一次带 NPC 队友的战斗，确认 `DPS` 页正常出数
2. 打开历史记录页，确认导入 / 导出 / 清空入口正常
3. 看一下主窗口版本号是否与当前发布版本一致
