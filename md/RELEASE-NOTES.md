## 主要更新

### 悬浮统计新增展示模式切换

- 悬浮窗现在支持 `Classic` 与 `Ikegami` 两种展示模式。
- 可以直接在设置里切换样式，不需要再手动改旧表格参数。
- `Classic` 与 `Ikegami` 会分别保存各自的悬浮窗尺寸与专属配置。

### 新增 Ikegami 横向条带式 overlay

- 新增横向条带式 Ikegami 悬浮窗样式。
- 姓名行位于色块上方，色块内只显示主数值，tooltip 直接挂在色块上。
- 去掉总 DPS 汇总卡与占比显性正文，最高伤害技能与数值可按需显示。
- footer 改为居中条带，并支持单独调整位置、间距、字号与透明度。

### 分享码、设置压缩与主题色优化

- 悬浮窗样式支持分享码导入导出，并会根据 `Classic` / `Ikegami` 标识自动导入到对应配置。
- Ikegami 专属设置新增尺寸、对齐、滚动条、透明度、字号等细项，并支持实时保存。
- 设置页卡片改成按实际内容高度自适应，说明文字与空白进一步压缩。
- 主题色调色板新增全局 `主题色透明度`；默认治疗职业主题色已同步到本轮确认结果。
