### 本次更新重点

#### 1. DPS 页面更完整

- 新增 `伤害量` 列
- `伤害量 / 职业 / 秒伤 / 死亡` 都可以在设置中单独隐藏
- 伤害数字改成中文单位显示，例如：
  - `9.68万`
  - `1.25亿`
  - `2.00兆`

#### 2. 悬浮面板更好用

- `DPS` 页签左键可折叠 / 恢复
- `DPS` 页签右键可直接打开 / 关闭设置窗口
- 新增悬浮表格设置：
  - `玩家列最小宽度`
  - `固定列宽`
  - `表格行高`

#### 3. 历史记录可以回看

- 历史记录现在会保存完整战斗结果
- 点击某条历史记录后，可以直接回看那场战斗的数据
- `DPS / HPS / 承伤 / 概览` 会一起切到对应记录
- 新战斗开始后，会自动切回实时数据

#### 4. 测试数据更实用

- `导入测试数据` 不再清空已有历史记录
- 重复导入时不会无限重复堆积同样的测试记录
- 3 条测试数据已经拆成不同场景，不再只是名字不同

#### 5. 统计范围更明确

- 当前版本只统计：
  - 自己
  - 当前队伍里的队友
- 队伍里的 NPC 队友也会被统计
- 队伍外玩家不会进入统计

#### 6. 插件入口补齐

- 已补上主窗口入口
- 修复“插件没有注册主 UI 回调”的校验提示

## 面向玩家的说明

如果你是普通玩家，这次更新最值得注意的是：

- 平时直接看悬浮 `DPS` 面板会更方便
- 现在可以看 `伤害量`
- 历史记录终于可以点进去回看
- 没有战斗时可以导入测试数据先检查面板效果

## 更新后建议

建议更新后优先做下面几件事：

1. 打开悬浮面板，确认 `DPS` 页面显示正常
2. 去设置里确认你想显示的列：
   - `职业`
   - `伤害量`
   - `秒伤`
   - `死亡`
3. 如果面板太挤，就调一下列宽和行高
4. 如果暂时没有战斗，就先导入测试数据，看看历史记录和界面效果

## 已知说明

- 当前版本优先保证“稳定能用”
- 某些更复杂的持续伤害 / 持续治疗补算逻辑仍在继续完善
- 如果你平时主要看当前队伍输出、承伤和历史记录，这一版已经可以正常使用
